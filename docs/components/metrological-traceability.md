# Metrological Traceability
**Scope**: DMP & DPP

**Status**: Completed

**Repository**: [https://codeberg.org/CE-RISE-models/metrological-traceability](https://codeberg.org/CE-RISE-models/metrological-traceability)

**Generated Schemas**: [https://ce-rise-models.codeberg.page/metrological-traceability/](https://ce-rise-models.codeberg.page/metrological-traceability/)

Reusable structures for documenting the reference basis of measured, calculated, method-defined, or boundary-defined values. The model covers measurands, units, standards, calibration chains, reference materials, measurement procedures, method references, and boundary-defined references.
